# Desafio-2.1-nodejs

## Como instalar

Clone o repositório
```bash
git clone https://github.com/Citeli-py/Desafio-2.1-nodejs.git
```

Crie um .env na pasta principal e coloque sua chave de api
```text
API_KEY=<SUA_CHAVE>
```

Instale as dependências
```bash
npm install
```

Execute aplicação
```bash
node conversor.js
```
